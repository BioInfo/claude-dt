# dt - Claude Code DevTools

Actionable intelligence from Claude Code session data.

```bash
dt ingest          # Parse sessions into DuckDB
dt report          # Weekly insights report
dt session list    # Recent sessions
dt antipatterns    # Detect anti-patterns
dt context         # Context efficiency analysis
dt tools           # Tool usage patterns
dt prompts         # Prompt quality analysis
dt query "SQL"     # Raw DuckDB queries
```
